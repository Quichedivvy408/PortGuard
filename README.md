## 🛠️ Usage Guide

PortGuard manages network ports to prevent unauthorized access. The application runs in the background of your system to maintain security.

*   **Launching the App:** Look for the PortGuard icon on your desktop or in your Start menu. Click the icon to start the interface.
*   **Initial Setup:** Upon the first launch, the program performs a diagnostic check of your network adapter. Give the application a moment to verify your connection status.
*   **Defining Rules:** PortGuard uses a list of rules to monitor incoming and outgoing data. You see a clear table within the main window showing active connections.
*   **Blocking Ports:** To block a specific port, click the Add button in the main window. Enter the number of the port you wish to close and toggle the switch to the On position.
*   **Saving Changes:** Click Apply to activate your new security rules.

## 📜 Understanding Your Dashboard

The application dashboard provides a view of your current network activity.
